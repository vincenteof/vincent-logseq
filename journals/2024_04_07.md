- 使用 [[nextjs]] 的 [[app router]]，[[rsc]] 可以直接做 data fetch，但是部分 client component 依旧需要引入客户端请求的方案，需要引入 fetch，react query，trpc 等依赖，这样一个项目会混杂太多不同的请求方案。
	- 可以使用 server action + react query 简化，但是 react 官方并不推荐用 [[rsa]] 做 data fetch。
