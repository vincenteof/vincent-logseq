		- 只有在使用无法在 R1CS 中直接表达的运算时才需要用 <-- 对信号赋值，但是最后一定也要通过 === 手动添加 constraint
			- `out[k] <-- (in >> k) & 1;`
	- 关于 constraint
		- circom 中的 constraint 是以下形式：
			- `A * B + c = 0`
			- A, B, C 都是信号的线性表达式
