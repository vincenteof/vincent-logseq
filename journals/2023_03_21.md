- what is a [[signal]]
	- useSignal 与 useState 的区别:
		- useState() => value + setter
		- useSignal() => getter + setter
		- getter 的好处是可以插入一些框架想要的逻辑，基本上和 vue 的 proxy 是一个道理，调用 getter 的地方会自动订阅 signal 的变更，这样可以自动去收集一个衍生计算的依赖，并且可以按需去 render 某些具体
